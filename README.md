# Valentine's Day Interactive Proposal

A fun, interactive Valentine's Day proposal page built with HTML, CSS, and JavaScript. Features playful animations, 3D effects, and a mischievous "No" button that won't let you say no easily!

## ‚ú® Features

- **3D Tilt Effect**: The card tilts based on mouse movement for an immersive experience.
- **Animated Background**: Floating blobs with soft blur effects create a dynamic backdrop.
- **Glassmorphism Design**: Modern glass-like card with backdrop blur and transparency.
- **Playful Interactions**:
  - "Yes" button with heartbeat animation that grows on each "No" hover.
  - "No" button that randomly moves around the card when hovered or touched.
- **Particle Effects**: Bubbles burst out when you click "Yes".
- **Responsive Design**: Works on desktop and mobile devices.
- **Smooth Animations**: CSS transitions and keyframes for fluid interactions.
